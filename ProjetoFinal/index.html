O que é Lorem Ipsum?

Quando desenvolvemos uma pagina, nem sempre iremos ter todo o conteúdo necessário para preenche-la, muitas vezes para conseguir fazer os testes será necessário adicionar um conteúdo falso, e é nesse momento que iremos recorrer ao Loren Ipsun

Loren ipsun, é um texto em latin, que podemos utilizar para criar conteúdo de teste dentro de nossas paginas.

Mas como eu vou fazer para escrever em latin?

A solução é simples, muitos sites oferecem ferramentas para retirar pequenos trechos desse texto em latin, então copiamos e colomos esses pequenos trechos dentro de nossos sites.

Um site que podemos utilizar para copiar trechos do Loren ipsun é http://www.lipsum.com/
